Routing Power Scale
How much of this ASN's routing table does the upstream carry, as seen from RIPE's global vantage points? Higher = more routes propagated = more active transit.
60–100 pts — Primary active transit. Carrying full or near-full routing table. This is a live, core transit provider.
25–59 pts — Backup / Secondary transit. Active but not primary. Typical for backup providers, failover paths, single-PoP connections, or regional carriers carrying a subset of prefixes.
0–24 pts — Likely historical. Session may be inactive, stale in RIPE's window, or only seen by a few route collectors. Treat with caution.
